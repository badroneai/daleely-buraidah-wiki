# مهمة إثراء بيانات التواصل — كافيهات بريدة

## المطلوب
البحث عن **رقم الهاتف** و**حساب الانستقرام** للكافيهات التالية باستخدام Google Maps وانستقرام.

## قواعد البحث
1. ابحث في Google Maps أولاً عن الكافيه بالاسم + بريدة
2. رقم الهاتف: من صفحة Google Maps مباشرة
3. الانستقرام: من صفحة Google Maps (قسم الروابط) أو بالبحث في انستقرام مباشرة
4. إذا لم تجد الكافيه أو لم تجد المعلومة، اترك الحقل فارغاً ""
5. لا تخمّن — فقط ضع ما تجده فعلياً

## صيغة التسليم
سلّم كل دفعة بصيغة JSON كالتالي:

```json
[
  {
    "slug": "اسم-الكافيه",
    "phone": "05XXXXXXXX",
    "official_instagram": "username_without_at_sign"
  }
]
```

## ملاحظات
- الهاتف بالصيغة السعودية: 05XXXXXXXX أو 01XXXXXXXX أو 920XXXXXX
- الانستقرام بدون @ — فقط اسم المستخدم
- إذا وجدت أكثر من رقم، ضع الرقم الرئيسي فقط
- إذا كان الكافيه له عدة فروع، ضع رقم الفرع المذكور في الحي
